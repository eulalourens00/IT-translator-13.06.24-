Miniature guide

"A"

ADSL (Asymmetric Digital Subscriber Line) — Технология, которая позволяет подключаться к интернету и обеспечивать высокоскоростную передачу данных через существующие телефонные линии

A.K.A (As Know As) - Аббревиатура "Известен как", "Другими словами"

API (Application Programming Interface) — Программный интерфейс приложения или интерфейс прикладного программирования. Простыми словами, это описание вариантов взаимодействия компьютерных программ между собой

Application — Прикладное программное обеспечение (компьютерная программа / мобильное приложение), предназначенное для выполнения определённых задач

Approve — Подтверждение, согласие, одобрение чего-либо после проведения предварительной проверки, модерации

ASP (Application Service Provider) — Организация - провайдер, предоставляющая услуги по аренде программных продуктов и инфраструктуры на базе периодических платежей с доступом к приложениям через Интернет или виртуальную частную сеть

"B"

Batch file — Пакетный файл, текстовый файл (MS-DOS, OS/2 или Windows), который содержит последовательность команд, служащих для исполнения командным интерпретатором. После запуска данного файла программа-интерпретатор (например, cmd.exe) читает его построчно и по порядку исполняет команды.

Bad — Почта, которая дошла получателю с ошибками, либо же неформальное название области, куда эта почта попадает (Badmail).

Badsector (Badblock) — Не читающийся/ненадежный кластер диска, который содержит в себе сбойные сектора, или помечен как таковой в структурах файловой системы.

Backup — Резервное копирование, т.е. создание резервной копии информации/проекта/сайта, для того, чтобы при каких-либо ошибках и сбоях иметь возможность восстановить данные.

Backlog — Перечень задач по функционалу в порядке приоритета для реализации в следующих версиях продукта.

Battery life — Продолжительность автономной работы.

Big data — Набор специализированных методов и инструментов, использующихся для хранения и обработки огромных объемов данных, т.е. таких, которые характеризуются «тремя V»: volume (объём), velocity (скорость прироста и обработки), variety (многообразие).

Bink (BinkleyTerm) — Бинком сейчас называют популярный IP-мейлер binkd. Это программа, разработанная для передачи почты и файлов в международной любительской сети Фидонет. В отличие от классических FTN-мейлеров, binkd работает по специально разработанному для него протоколу binkp, используя в качестве транспорта TCP/IP. Таким образом, binkd позволяет передавать почту как через Интернет, так и по локальной сети. (Binkd распространяется под лицензией GNU GPL v.2.)

Blockchain — Методика построения непрерывной последовательной цепочки блоков (связного списка), которые содержат определенную информацию. Внесение изменений в информацию существующих блоков очень проблематично, т.к., чаще всего, копии цепочек блоков хранятся на множестве разных компьютеров независимо друг от друга. Кроме того, если необходимо внести какие-либо изменения в информацию одного из блоков, придётся редактировать и все последующие блоки.

BSoD (Blue Screen of Death) — «Синий экран смерти», уведомление о крупной ошибке системы в некоторых версиях OS Windows. Обычно требует перезагрузки системы.

"C"

CD-R (Compact Disk-Recordable) — Стандарт компакт-дисков и дисководов, позволяющий осуществлять не только чтение, но и однократную запись данных на CD.

CGI (Common Gateway Interface) — Интерфейс, позволяющий взаимодействовать программам клиента с программами, запущенными на сервере. Интерфейс прикладного программирования, используеммый в Internet. CGI — спецификация, определяющая взаимодействие внешней программы создания динамических www-документов и http-сервера.

Client technology — Метод распределения конкретной задачи между несколькими компьютерами.

CMOS (Complementary Metal-Oxide Semiconductor) — Микросхема, содержащая сведения о параметрах устройства (см. BIOS).

Color Depth — количество бит, приходящихся на один пиксель (bpp), чем больше это число, тем больше цветов может использовать изображение.

Configuration — Конфигурация, совокупность программных и аппаратных средств компьютера.

Counter — Счетчик посещений на Web-страницах.

CPS (Check Clicks per Second) — Под этим сокращением понимают скорость передачи/приема данных модемом. Измеряется в битах в секунду.

Corrupt — Повреждение, дефект.

CVS (Concurrent Versions System)— Система контроля версий. С помощью CVS можно обратиться к старым версиям для выяснения причины ошибки.


"D"

DAE (Digital Audio Extraction) — Процесс копирования аудиоинформации на жесткий диск компьютера.

DDE (Dynamic Data Exchange) — Динамический обмен данными. Стандартный программный протокол для взаимодействия между процессами.

Decrement — Уменьшение значения переменной на заданное значение (обычно, на единицу).

Deluxe — Расширенная версия какого-либо программного пакета, включающая дополнительные программы или возможности.

Denial of Service — Отказ сервера в обслуживании, невозможность обработки информации из-за множества поступающих запросов.

Demo (Demonstration») — Демонстрационная версия чего-либо с ограниченным функционалом.Также это жанр компьютерного искусства, заключающийся в создании мультимедийной презентации.

Destructor — Специальный тип метода класса, который выполняется при удалении объекта класса. Деструкторы созданы для очистки памяти после инициализации класса с помощью конструкторов.

Domain — Уникальный онлайн-адрес, место его размещения в Интернете.

Drop — Подразумевается обычно удаление чего-либо (базу данных, приложение, таблицу и т.д.).

DSP — (Digital Signal Processor) — Специализированная микросхема, отвечающая за обработку сигналов. Используются для преобразования аналоговых сигналов в цифровые и наоборот.

Dump — Файл, содержащий резервную копию системы/памяти компьютера/базы данных, актуальную на момент его создания. В разработке их применяют для создания и поддержания в актуальном состоянии системы бэкапа.

"E"

Enable — Включить, разрешить использование подключенного устройства или системной функции.

Encapsulation — Упаковка данных или функций в единый компонент. Процесс скрытого хранения деталей реализации объекта от пользователей и предоставление им для работы интерфейсов классов.

EPROM (Erasable Programmable Read Only Memory) — Программируемая микросхема памяти.

EPS (Encapsulated PostScript) — Формат для описания текстовых и графических данных. Разработан фирмой Adobe. Использует PostScript.

Exchange — Обмен данными.

Extension — Расширение имени файла. Расширение отделяется от собственно имени файла точкой.

"F"

Fail — Отсутствие данных.

Fire wall — (Брандмауер) Программа-антивирус, обеспечивающая обнаружение и защиту компьютера от вирусов и вредоносных ПО.

FidoNet — Глобальная любительская некоммерческая не анонимная компьютерная сеть (аналог интернета), созданная по технологии «из точки в точку».

Forward – Переслать поступившее письмо другому пользователю.

"G"

GFLOPS (Billions of FLOating Point Operations Per Second (GigaFlops)) - Миллиард операций с плавающей точкой в секунду. Один из показателей скорости обработки информации.

GPRS (General Packet Radio Service)— Расшифровывается как пакетная передача данных по радиоканалу.

Gopher — Распределенная система экспорта структурированной информации. Сервис прямого доступа и требует, чтобы и сервер, и клиент были полноценно подключены к Интернету.

Guest Book — Программа, запущенная на стороне клиента. Позволяет посетителям высказать свое мнение о страничках и послать его авторам.

GUI (Graphical User Interface) — Графический пользовательский интерфейс.

"H"

HDSL (High-bit-rate DSL ) — Технология высокоскоростной синхронной передачи данных по витой паре на расстоянии до 6 км со скоростью до 2, 048 Мбит/c.

Headless — Компьютер, который работает без устройства для вывода визуала, т.е. дисплея/монитора. Это может быть, к примеру, сервер. Безголовая система обычно управляется через сетевое соединение.

Htaccess — файл, позволяющий задать множество полезных настроек, используемых при выдаче файлов клиенту. С помощью этого файла Вы можете: запретить доступ к файлу, всем файлам, группе файлов по маске.

"I"

IIS (Internet Information Server) — Набор интегрированных сервисов, позволяющих размещать, управлять и организовывать доступ к веб-серверам, создавать и управлять веб-приложениями.

Implementation — В объектно-ориентированном программировании термин обозначает практическую реализацию методов.

Increment — Уменьшение значения переменной на заданное значение (обычно, на единицу).

Incorrect — Некорректная операция, неправильный параметр и т. д.

Insert — Вставка, стандартный пункт меню программ Windows.

I/O (Input / Output) — Устройство ввода-вывода информации.

IRQ (Interrupt Request) — Прерывание, специальный канал связи для передачи сигналов между подключенным к компьютеру устройством и центральным процессором.

ISDN (Integrated Services Digital Network) — Цифровая сеть комплексных услуг — высокоскоростная линия, используемая для удаленного доступа к сети.

"J"

Join — Использовать, задействовать.

Jumper — Переключатель, соединяющей контакты на материнской плате.

"K"

Keep — Держать, сохранять установки.

KISS (keep it simple, stupid) — Принцип: «не усложняй».

Kludge — Управляющие строки, которые содержат техническую информацию о письме, скрытую по умолчанию при просмотре в редакторе.

"L"

Lag — Торможение или медленная работа какой-либо системы.

LAN (Local Area Network) — Локальная компьютерная сеть.

Leech — Пользователь файлообменной сети, который пользуется ресурсами других, ничего не загружая взамен.

Log — Текстовый файл отчета, в который записываются все действия, выполняемые программой, а также их результаты.

"M"

Mailing List — Список почтовой рассылки, по которому программа автоматически рассылает письмо.

Malicious software — Вредоносное программное обеспечение, созданное с целью нанести урон компьютеру с его содержимым.

Master — Основное устройство.

Merge — Объединение нескольких веток кода или данных таблиц.

Microsoft FrontPage Server Extension — дополнительная возможность веб-сервера, позволяющая использовать инструмент создания и ведения сайта. Прост в использовании, позволяет создавать сложные динамические решения.

MHS (Message Handling System) — Система управления сообщениями, совместно обеспечивающая функционирование электронной почты.

"N"

N/A (Not available) — Сокращение, используемое при общении в интернет и в некоторых программах для обозначения статуса пользователя на данный момент. Обозначает: нет данных, не доступен.

Netscape Navigator (NN) — Один из броузеров, программ для просмотра веб-страниц.

Netiquette — Сетевой этикет, правила поведения в Интернете.

NSFW (Not safe for work) — Небезопасно для работы.

"O"

OCR (Optical Character Recognition) — Система автоматического распознавания текста, введенного в компьютер через сканер в виде графического изображения.

ODBC (Open DataBase Connectivity) — Открытая связь с базой данных — технология, предоставляющая стандартный интерфейс доступа к базам данных разных типов.

OEM (Original Equipment Manufacturer) — Способ продажи программного обеспечения вместе с готовыми компьютерами, а также с отдельными комплектующими.

OLE — Технология ObjectLinkingandEmbedding, дающая возможность вносить изменения в данные, которые были созданы в иной программе, не покидая основной редактор.

OPML — Один из форматов предназначенных для хранения списка источников новостей на веб-сайте.

Overclocking — Разгон процессора и ряда других компьютерных устройств, установка повышенной рабочей частоты с целью увеличения производительности.

"P"

PageRank (PR) — Алгоритм определения популярност веб-сайта.

Patch — Программа, исправляющая ошибки в какой-либо программе или добавляющая в нее новые возможности. См. Update. Кроме того, в терминологии мультимедиа патчами называются внешние библиотеки инструментов, загружаемые в память звуковой карты при воспроизведении MIDI-музыки.

PCMCIA (Personal Computer Memory Card International Association) — Спецификация дополнительных устройств для портативных компьютеров, в том числе — модемов, карт оперативной памяти и т. д.

Performance — Производительность, скорость работы компьютера.

PhpMyAdmin — Утилита, написанная на PHP для удобного администрирования MySQL через веб.

Plug-in — Подключаемая программа, выполняющая какие-либо дополнительные функции в главной программе.

Prompt — Приглашение.

"Q"

QA (Software Testing and Quality Assurance) — Специалист по обеспечению качества программного обеспечения.

"R"

R&D (Research and Developmen) — Исследования и разработки.

Refresh — Частота обновления кадров на экране монитора.

Run — Запустить программу.

"S"

Scheduler — Планировщик задач. Программа, позволяющая запускать другие программы в заданное пользователем время.

SMART (Self-Monitoring Analysis and Reporting Technology) — Система оперативной самодиагностики, встроенная в жесткие диски последних моделей.

S/PDIF — Цифровой вход/выход на звуковых картах высшего класса.

Sux — Крайняя степень неодобрения чего-либо, уничижительное определение.

SQL (Structured Query Language) — Язык структурированных запросов, используемый для работы с базами данных.

"T"

Template — Шаблон.

Temporary — Временный.

Tip — Совет, подсказка.

TLS — Протокол, используемый для аутентификации и защиты данных.

Tools — Приспособления, инструменты.

Trialware — Пробная версия коммерческой программы, работающая в течение небольшого срока после инсталляции (как правило, 14-30 дней).

Troubleshooting — Рекомендации по решению проблем, средства для устранения неисправностей.

TWAIN (Technology Without Any Interesting Name) — Программный интерфейс для подключения ряда внешних устройств ввода.

Tweak — Небольшая утилита для подстройки скрытых, недокументированных параметров программы или устройства.

"U"

Undo — Отменить действие, операцию.

Usenet — система телеконференций, организованная в форме каталога, формируемого на основе групп новостей по определенным предметным новостям.

Use Case — Техника описания взаимодействия пользователя с продуктом через сценарии.

UPS — Источник бесперебойного питания.

"V"

Virtual Private Server — Услуга, в рамках которой пользователю предоставляется виртуальный выделенный сервер.

"W"

Warez — интеллектуальная собственность или ПО, свободно распространяемое в Интернете, незаконным путем с нарушением прав правообладателя.

Worm — Вредоносная программа. Один из типов компьютерных вирусов.

WYSIWYG — визуальный редактор.

"X"

XML — Расширяемый язык разметки гипертекста.

"Y"

Y2K — 2000-й год. Чаще всего эта аббревиатура употребляется в связи с ошибкой 2000-го года — невозможностью некоторых старых программ и компьютеров правильно интерпретировать смену дат при переходе в новое тысячелетие.

"Z"

ZIP-code — Почтовый индекс.

Zoom — Увеличение/уменьшение экранного изображения (соответственно — Zoom In и Zoom Out).
